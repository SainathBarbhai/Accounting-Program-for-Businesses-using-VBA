# Accounting-Program-for-Businesses-using-VBA
Created a program using Visual Basic with Applications (VBA) that can help a business maintain its finances.

The program is equipped to perform the following-

Maintain a record of...
1. Real-time purchase history, and inventory data, 
2. Payroll for employees, 
3. Generate Purchase Orders (POs) and invoices, 
5. Vendor and Customer information,
4. Assets and Liabilities to produce a Balance Sheet and P&L statement for fiscal year-end tax purposes
